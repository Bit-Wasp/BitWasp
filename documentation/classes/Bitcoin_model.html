update_credits()

update_credits( $updates) : bool

Update Credits

Called by Blocknotify when crediting accounts, and by bitcoin::cashout() when sending accounts. And by escrow_model::pay() to issue payment. And orders::place() to deduct the buyers account. Requires an array, where each update is contained as an entry in that array. Often only suppling one update. The value can be negative. Called by Wallet Notify when debiting accounts. Eg: $updates = array('0' => array('user_hash' => '...', 'value' => '...'), '1' => array('user_hash' => '...', 'value' => '...'));

Parameters

$updates
